# Arduino-fan-game
A playable Star Wars game, run by an Arduino and a few additional push buttons. 
The main code folder contains V1 and V2 codes, you oled display may not support one or the other. so test which one your OLED supports. Install those libraries from the library manager of your IDE or just download the attached libraries and put them in your IDE's library folder. The directory looks something like this: C:\Users\rohan\Documents\Arduino\libraries. Find yours in the Documents tab in File Manager.
